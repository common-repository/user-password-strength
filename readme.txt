Enforce a specific individual password strength. Set password length, number of symbols, numbers and lower/upper characters for your minimum user-passwords specification

== Description ==

Each staff-member or user has the opportunity to change his own password in Wordpress.
Wordpress contains a visual indicator for your password strength, but you still allowed to save a weak password.
This could be a problem for sensitive websites. So make sure that your staff or users take a save password with specific password strength, which you can determine.

With this Plugin the Administrator can set a minimum length for passwords.
It is also possible to set a minimum number of symbols, numbers and/or lower/upper characters for passwords.

The Plugin is WPML ready.

== Installation ==

1. Upload the 'password-strength' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your required password strength in Settings -> Password Strength
